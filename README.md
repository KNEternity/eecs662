## How to run Haskell Code 
I initally had some trouble running haskell code because it's not the same as running a python program. I used GHCI, which lets you paste haskell functions into the terminal for some quick debugging. I have written most test cases for all of the functions to ensure they run. 

Visit this website to install Haskell -> https://www.haskell.org/ghcup/
